# Student Result Management System
**Prepared by:** Hariom (25MIM10019)


# Project Overview
This is a simple console-based Python project designed to manage student marks.  
It allows the user to enter student information, calculates the total marks, percentage, and grade, and stores the data in a CSV file for future use.

This project demonstrates:
- Python basics
- File Handling (CSV)
- Data Processing using Pandas
- Simple Menu-Driven Program


# Tools & Technologies Used
- **Python 3.x**
- **Pandas** library (for data storage and calculations)


# Features
✔ Add student details: Roll No, Name, and Marks in 3 subjects  
✔ Auto-calculate:
- Total Marks
- Percentage
- Grade  
✔ View all saved student records  
✔ Stores data automatically in students.csv
